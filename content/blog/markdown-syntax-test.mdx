### JavaScript

```javascript
// 一个简单的 JavaScript 函数
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

console.log(fibonacci(10)); // 输出: 55
```

### 块级公式

高斯积分：

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

欧拉恒等式：

$$
e^{i\pi} + 1 = 0
$$

麦克斯韦方程组之一：

$$
\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}
$$

公式和文字混合出现：$e^{i\pi} + 1 = 0$ 这是一个公式。

## 长段落测试

这是一段较长的文字，用于测试段落的行间距和阅读体验。良好的排版应该让读者感到舒适，不会因为行距太窄而感到拥挤，也不会因为行距太宽而难以连贯阅读。一般来说，正文的行高设置为 1.5 到 1.75 之间是比较合适的。

另外，段落之间也需要适当的间距来区分不同的内容块。这有助于读者理解文章的结构，知道哪些内容是属于同一个主题的。在技术文档中，清晰的结构尤其重要，因为读者通常需要快速找到他们需要的信息。

最后，字体大小也很重要。太小的字体会让读者感到疲劳，太大的字体则会显得不够专业。通常，正文字体大小设置为 16px 到 18px 是比较合适的选择。
